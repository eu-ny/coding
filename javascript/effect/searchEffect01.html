CSS 속성 검색하기

CSS 속성 갯수 :
  • accent-color : accent-color는 요소의 강조 색상을 지정합니다.
  • align-content : align-content는 콘텐츠 사이와 콘텐츠 주위 빈 공간을 플렉스 박스'의 교차축 또는 그리드의 블록 축을 따라 배치하는 방식을 결정합니다.
  • align-items : align-item은 콘텐츠 아이템의 내부 상하관계 정렬 상태를 설정합니다.
  • align-self : align-self는 개별적 콘텐츠 아이템의 정렬 상태를 설정합니다.
  • all : all은 요소의 모든 속성을 초기화합니다. 또는 상속을 설정합니다.
  • animation : animation은 애니메이션과 관련된 속성을 일괄적으로 설정합니다.
  • animation-delay : animation-delay는 애니메이션 지연 시간을 설정합니다.
  • animation-direction : animation-direction는 애니메이션 움직임 방향을 설정합니다.
  • animation-duration : animation-duration는 애니메이션 움직임 시간을 설정합니다.
  • animation-fill-mode : animation-fill-mode는 애니메이션이 끝난 후의 상태를 설정합니다.
  • data-type="animation"animation-iteration-count : animation-iteration-count는 애니메이션의 반복 횟수를 설정합니다.
  • animation-name : animation-name은 애니메이션 Keyframe 이름을 설정합니다.
  • animation-play-state : animation-play-state는 애니메이션의 진행 상태를 설정합니다.
  • animation-timeline : animation-timeline은 요소에 적용할 스크롤 애니메이션을 설명하는 하나 이상의 @scroll-timeline at-rules 이름을 지정합니다.
  • animation-timing-function : animation-timing-function은 애니메이션 움직임의 속도를 설정합니다.
  • appearance : appearance는 webit 계열의 브라우저의 type="search" 필드의 둥근 테두리 값이나 reset 효과를 나타내는 버튼을 삭제하는 데 사용할 수 있습니다.
  • aspect-ratio : aspect-ratio는 요소의 크기를 비율대로 조정할 수 있게 합니다.
  • backdrop-filter : backdrop-filter는 요소 뒤 영역에 흐림이나 색상 시프트 등 그래픽 효과를 적용할 수 있는 속성입니다.
  • backface-visibility : backface-visibility는 요소를 입체적으로 볼 때 요소의 뒷면이 사용자에게 보이도록 하는 속성입니다.
  • background : background는 백그라운드 속성을 일괄적으로 설정합니다.
  • background-attachment : background-attachment는 배경 이미지의 고정 여부를 설정합니다.
  • background-blend-mode : background-blend-mode는 백배경을 혼합했을 때 그래픽 효과를 설정합니다.
  • background-clip : background-clip은 백그라운드 이미지의 위치 기준점을 설정합니다.
  • background-color : background-color는 백그라운드 색을 설정합니다.
  • background-image : background-image는 백그라운드 이미지 속성을 설정합니다.
  • background-origin : background-origin는 백그라운드 이미지의 위치 기준점을 설정하기 위한 속성입니다.
  • background-position : background-position은 백그라운드 이미지의 위치 영역을 설정합니다.
  • background-position-x : background-position-x는 백그라운드 이미지의 X축 위치 영역을 설정합니다.
  • background-position-y : background-position-y는 백그라운드 이미지의 y축 위치 영역을 설정합니다.
  • background-repeat : background-repeat는 백그라운드 이미지의 반복 여부를 설정합니다.
  • background-size : background-size는 백그라운드 이미지 사이즈를 설정합니다.
  • block-size : block-size는 쓰기 모드에 따라 요소 블록의 가로 또는 세로 크기를 정의합니다.
  • border : border는 테두리 속성을 일괄적으로 설정합니다.
