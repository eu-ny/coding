indexOf()를 통해 검색 사이트 제작하기

지난 시간에는 함수와 반복문을 이용하여 퀴즈 사이트를 제작해보았습니다. 이번 시간에는 문자열 메서드 중 하나인 indexOf() 메서드를 통해 검색 사이트를 제작해볼까요?!
먼저 indexOf() 메서드에 대해 알아보겠습니다.

#1. indexOf()

indexOf() 메서드는 문자열에서 특정 문자의 위치를 찾습니다.

"문자열".indexOf(검색값)
"문자열".indexOf(검색값, 위치값) 

const str1 = "javascript reference"
            
const currentStr1 = str1.indexOf("javascript"); // 0
const currentStr2 = str1.indexOf("reference"); // 11 (11번째 자리 : 위치 값 출력)
const currentStr3 = str1.indexOf("j");        // 0
const currentStr4 = str1.indexOf("a");        // 1 (중복일 때 가장 처음 것만 출력)
const currentStr5 = str1.indexOf("v");        // 2
const currentStr6 = str1.indexOf("jquery");   // -1
const currentStr7 = str1.indexOf("b");        // -1
const currentStr8 = str1.indexOf("javascript", 0);   // 0
const currentStr9 = str1.indexOf("javascript", 1);   // -1
const currentStr10 = str1.indexOf("reference", 0);   // 11
const currentStr11 = str1.indexOf("reference", 1);   // 11
const currentStr12 = str1.indexOf("reference", 11);  // 11
const currentStr13 = str1.indexOf("reference", 12);  // -1

! point ! 만약 선택한 문자가 여러개일 경우 가장 첫번째 문자만 출력합니다.

#3. JAVASCRIPT : 스크립트 짜기

indexOf()와 조건문을 통해 스크립트를 완성해보자!
코드의 주석과 함께 스크립트를 이해해보도록 합니다:) 

//선택자
const searchBox = document.querySelector(".search__box input");       //검색 영역
const searchList = document.querySelectorAll(".search__list ul li");     //목록 리스트

searchBox.addEventListener("keyup", () => {
    const searcWord = searchBox.value; //사용자가 입력한 키워드

    searchList.forEach(el => {
        const cssName = el.dataset.name;

        //0이 되었을 때 false가 된 값을 보이게 한다. keyup을 했기 때문에 한 글자만 작성해도 실행.
        if(cssName.indexOf(searcWord)){
            el.classList.add("hide");      //검색한 단어 외의 단어 숨김
        } else {
            el.classList.remove("hide");   //검색한 단어를 보이게 함 
        }
    });
});

! point ! indexOf()값이 0(=false)이 되는 것만 hide를 지워준다.
