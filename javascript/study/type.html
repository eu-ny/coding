데이터 타입

데이터 타입은 변수에 저장되는 데이터의 유형으로 원시 데이터 타입과 객체 데이터 타입으로 나눌 수 있다.

#1. 숫자(number) 데이터

숫자 데이터는 정수, 소수점 등을 표현 할 수 있다.

let table = 1;
let table2 = 1.5;

#2. 문자(string) 데이터

문자 데이터는 '' 또는 "" 로 표현할 수 있다. 보통 문자 데이터를 사용할 땐 작은 따옴표 '' f를 많이 표현
문자안에 문자가 들어갈 땐 서로 다른 따옴표를 사용해주어야 한다.

let table = "문자와 '문자"
let table2 = '문자와 "문자"'

#3. 논리(boolean) 데이터

논리 데이터는 true와 false의 값을 표현한다.
논리 데이터에서 0값은 false를 의미하며 0값 이외의 숫자 및 문자 값은 true를 의미한다.
Boolean()명령어 = 변수의 값을 true나 false로 변환

let table = (5 > 2); // true
let table2 = (5 < 2); // false
let table3 = 0; // false
let table3 = 1; // true

#4. 배열( [] ) 데이터

배열 데이터는 순서가 있는 데이터 집합이며, 변수 하나에 여러개의 값을 순차적으로 저장시킬 수 있다.

let table = ["자바스크립트","100"];
let table2 = [100, 200];

#5. 객체( {} ) 데이터

객체 데이터는 순서가 있는 데이터 집합이며, 변수 하나에 여러개의 값을 순차적으로 저장시킬 수 있다.

let table = ["자바스크립트","100"];
let table2 = [100, 200];
