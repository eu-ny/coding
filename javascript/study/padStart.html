자바스크립트 : 문자열 메서드 : padStart() / padEnd()

이번 시간에는 문자열 메서드인 padStart() / padEnd()에 대해 알아보겠습니다! 😁

#1. padStart( )

padStart() 메서드는 주어진 길이에 맞게 앞 자리에 문자열을 채우고, 새로운 문자열을 반환합니다.

! padStart( ) 사용 방법 !
"문자열".padStart(길이(자리수));
"문자열".padStart(길이(자리수), "추가 할 문자열"); 

const str1 = "456";

const cirrentStr1 = str1.padStart(1, "0");      //456
const cirrentStr2 = str1.padStart(2, "0");      //456
const cirrentStr3 = str1.padStart(3, "0");      //456 : 기존 문자열에서 자리수가 늘어나지 않았기 때문에 문자열이 추가되지 않는다.
const cirrentStr4 = str1.padStart(4, "0");      //0456 : 자릿수보다 클 경우 앞에 0이 붙는다.
const cirrentStr5 = str1.padStart(5, "0");      //00456 : 자릿수보다 클 경우 앞에 0이 붙는다.
const cirrentStr6 = str1.padStart(6, "0");      //000456 : 자릿수보다 클 경우 앞에 0이 붙는다.
const cirrentStr7 = str1.padStart(6, "1");      //111456 : 자릿수보다 클 경우 앞에 1이 붙는다.
const cirrentStr8 = str1.padStart(6, "12");     //121456 : 자릿수보다 클 경우 앞에 121이 붙는다.
const cirrentStr9 = str1.padStart(6, "123");    //123456 : 자릿수보다 클 경우 앞에 123이 붙는다.
const cirrentStr10 = str1.padStart(6, "1234");  //123456 : 자리수보다 문자열이 길기 때문에 문자열의 마지막 값은 출력되지 않는다.
const cirrentStr11 = str1.padStart(6);          //   456 : 뒤에 문자열을 입력하지 않으면 앞의 자리수만큼 공백 처리 된다. : 456은 자릿수가 3이니 남은 3만큼의 공백이 생김

! point ! 문자열을 추가했을 때 문자열의 자리수보다 문자열이 긴 경우 추가 된 문자열의 마지막 값은 출력되지 않는다.
이 때 만약 자리수는 늘어났는데 추가할 문자열이 없을 경우 공백처리한다.
기존 문자열에서 길이를 추가했을 때 자리수가 늘어나지 않았을 경우 문자열은 추가되지 않는다.

#2. padEnd( )

padEnd() 메서드는 주어진 길이에 맞게 뒷 자리에 문자열을 채우고, 새로운 문자열을 반환합니다.

! padEnd( ) 사용 방법 !
"문자열".padEnd(길이);
"문자열".padEnd(길이, "문자열"); 

const str1 = "456";

const cirrentStr1 = str1.padEnd(1, "0");     // 456
const cirrentStr2 = str1.padEnd(2, "0");     // 456
const cirrentStr3 = str1.padEnd(3, "0");     // 456
const cirrentStr4 = str1.padEnd(4, "0");     // 4560
const cirrentStr5 = str1.padEnd(5, "0");     // 45600
const cirrentStr6 = str1.padEnd(6, "0");     // 456000
const cirrentStr7 = str1.padEnd(6, "1");     // 456111
const cirrentStr8 = str1.padEnd(6, "12");    // 456121
const cirrentStr9 = str1.padEnd(6, "123");   // 456123
const cirrentStr10 = str1.padEnd(6, "1234");  // 456123
const cirrentStr11 = str1.padEnd(6);          // 456___ (공백이 티가 안나서 임시로 하이픈으로 대체!)

! point ! 문자열을 추가했을 때 문자열의 자리수보다 문자열이 긴 경우 추가 된 문자열의 마지막 값은 출력되지 않는다.
이 때 만약 자리수는 늘어났는데 추가할 문자열이 없을 경우 공백처리한다.
기존 문자열에서 길이를 추가했을 때 자리수가 늘어나지 않았을 경우 문자열은 추가되지 않는다.
