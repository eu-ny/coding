자바스크립트 : JQuery로 라이브러리

이번 시간에는 자바스크립트의 라이브러리인 jquery에 대해 알아보겠습니다.😆

#1. JQuery란?

JQuery로 HTML의 요소들을 쉽게 조작하고 편리하게 사용할 수 있게 만든 라이브러리로 Javascript를 미리 작성해둔 것입니다.
Javascript로 길고 복잡하게 써야 하는 것을 JQuery로 보다 짧고 직관적으로 쓸 수 있다.

📖 jquery 사용 방법 📖
제이쿼리는 자바스크립트 라이브러리이므로, 제이쿼리 파일은 자바스크립트 파일(.js 파일) 형태로 존재합니다.
제이쿼리를 사용할 때는 제이쿼리 공식 사이트에서 제이쿼리를 다운로드 받거나, CDN을 이용하면 됩니다.
ex) 구글의 CDN <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

#2. jquery 기본 문법

$(선택자).동작함수();

$ 기호는 제이쿼리를 의미하고, 제이쿼리에 접근할 수 있게 해주는 식별자입니다.
선택자를 이용하여 원하는 HTML 요소를 선택하고, 동작 함수를 정의하여 선택된 요소에 원하는 동작을 설정합니다.

#3. jquery 활용 예시

)//Javascript
document.querySelector(".slider").innerHTML = "image";

//Jquery
$(".slider").html("image");

💡 아래의 실제 제이쿼리를 적용한 코드를 자바스크립트와 비교하면서 더 자세히 알아봅시다! 

//자바스크립트
sliderInner = document.querySelector(".slider__inner");
slider = document.querySelectorAll(".slider")
let currentIndex = 0;
        
sliderInner.style.transition = "all 0.6s";
setInterval(() => {
    currentIndex = (currentIndex + 1) % slider.length;
        
    sliderInner.style.transform = "translateX("+ -800 * currentIndex +"px)";
}, 2000);

아래 코드처럼 제이쿼리는 더 간편하게 스크립트를 작성할 수 있습니다. 

//제이쿼리
let currentIndex = 0;
    
setInterval(() => {
    currentIndex = (currentIndex + 1) % $(".slider").length;
        
    $(".slider__inner").css("position", "relative");
    $(".slider__inner").animate({ left: -800 * currentIndex }, 600)
}, 2000);
