자바스크립트 전역변수 / 지역변수

변수는 함수블록 {}을 기준으로 변수의 선언 위치에 따라 전역변수와 지역변수로 나누어 진다.

전역변수 : 전역 변수는 함수 블록 {} 밖이나 안에서 자유롭게 사용 가능
지역변수 : 지역 변수는 함수 블록 {} 내에서만 사용 할 수 있다.

▶︎ 전역변수와 지역변수 선언 위치

전역 변수 지역 변수
var 변수;
      function 함수 () {
} 		function 함수 () {
     var 변수;
 }

01. 전역변수 예시 문제

kor 변수의 값을 90에서 100으로 바꾸기 

{
    var kor = 90;
    function getScore() {
       kor = 100; //전역변수
       console.log(kor);
    }
    getScore();
    console.log(kor);
}
결과 보기
100
100

kor 변수는 함수 블록 안에 있기 때문에 전역 변수이다.
따라서 getScore() 함수를 호출하게 되면 kor 변수의 값이 90에서 100으로 바뀌게 된다.

02. 전역변수 예시 문제

kor 변수의 값을 90에서 100으로 바꾸기

const func(){
     document.write("실행문");
}
func(); 

{
    var kor = 90;
    function getScore() {
       var kor = 100; //지역변수
       console.log(kor);
    }
    getScore();
    console.log(kor);
}
결과 보기
100
90

kor 변수가 그대로 90으로 출력 된 이유는 함수 블록 {} 안에서 kor 변수를 var로 다시 정의했기 때문이다.
이 때 기존 kor 변수와 함수 블록 {} 안의 kor변수는 서로 다른 별도의 함수가 되기 때문에 변수의 값이 바뀌지 않은 것이다.

함수 블록 안의 지역 변수는 반드시 var로 정의해야 한다. 또한 지역 변수는 함수 블록 밖에서는 사용할 수 없다.
이렇게 블록{}에 의해 변수의 범위가 달라지는 것을 변수의 스코프(Scope)라고 한다.

함수 레벨 스코프와 블록 레벨 스코프

const func(){
     document.write("실행문");
}
func(); 

{
    var kor = 90;
    function getScore() {
       var kor = 100; //지역변수
       console.log(kor);
    }
    getScore();
    console.log(kor);
}
결과 보기
100
90
