레이아웃05

부모컨테이너를 만들어서 가운데 영역을 설정한다.

float을 이용한 반응형 레이아웃

clearfix를 이용하여 레이아웃을 고치기 위해서는 고치려는 요소의 부모요소에게 clearfix를 선언해주면 된다.
div class="container clearfix"
또한 이러한 속성을 효율적으로 주기 위해서는 html 마크업을 깔끔하게 선언할 줄 알아야 한다. 때문에 먼저 레이아웃 분석 및 요소의 이름을 잘 선언하는 것이 중요하다.

    /* float으로 인한 영역 깨짐 방지법 
    1. 깨지는 영역에 clear:both를 설정한다.
    2. 부모 박스 영역에 overflowf: hidden을 설정한다. - 애니메이션 효과가 적용되지 않는다.
    3. clearfix를 설정한다. - 망가진 레이아웃 부모박스에 클래스로 이름을 설정해준다.
    */
    /* clearfix  */
    .clearfix::before {
        zoom: 1;
    }

    .clearfix::before .clearfix::after {
        content: '';
        display: block;
        line-height: 0;
    }

    .clearfix::after {
        clear: both;
    }
