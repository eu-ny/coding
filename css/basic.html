CSS 단위

px

해상도에 따라 상대적으로 달라지는 기본 단위

%

부모 요소의 글자 크기를 100%기준으로 계산한 % 단위
특별한 설정이 없다면 16px = 100% / 24px = 150%가 된다.

em

부모 요소의 글자 크기를 100%기준으로 계산한 100분의 1단위
특별한 설정이 없다면 16px = 1em / 24px(16px x 1.5 = 24px) = 1.5em가 된다.

rem

em과 비슷하지만 부모 요소가 아닌 최상위 요소의 크기를 100%기준으로 계산한다.
r = root 즉 html요소를 뜻하므로 html 요소에서 지정한 글자 크기가 1rem이 된다.

vw, vh

vw : 뷰포트의 너비값의 100분의 1 단위
vh : 뷰포트의 높이값의 100분의 1 단위
