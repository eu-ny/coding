CSS : 색상 표현 방법

CSS에서는 색상을 설정하는 총 7가지의 방법이 있습니다. 먼저 기존 CSS에 있던 방법 3가지, CSS3에서 추가된 4가지 방법이 있습니다.
이번 시간에는 이러한 CSS의 색상 표현 방법에 대해 알아보겠습니다!😄

🔶 CSS 색상 표현 방법 🔶

#1. 색상 이름으로 표현

red, pink와 같이 색상의 이름을 나타내는 방법으로, 약 140가지의 색상 이름들이 브라우저에 미리 정의되어 있습니다. 

p {background-color: orange;}

orange

#2. RGB값으로 표현

rgb(red, green, blue)와 같은 형식으로 색을 표현합니다.
여기서 red, green, blue 각각의 변수에는 0에서 255사이의 정수를 입력하거나 백분율 값을 지정하여 색상을 설정합니다. 

p {background-color: rgb(255, 165, 0);}
p {background-color: rgb(100%, 0%, 0%);}

orange

red

#3. HEX(16진수)값으로 표현

hex(16진수) 색상값은 RGB 색상값을 각각 16진수로 변환한 것입니다.
#rrggbb = rr: red / gg: green / bb : blue로 00에서 ff까지의 범위 값입니다. 

p {background-color: #FFA500;}

orange

🔶 CSS3에서 추가된 색상 표현 방법 🔶

#1. RGBA 색상값으로 표현

RGBA 색상값은 RGB 색상값에 알파 채널 값을 더한 색상값입니다.
여기서 알파 채널(alpha channel)이란 색상의 투명도를 나타내는 채널입니다. 알파 채널 값은 완전한 투명 상태인 0.0부터 투명도가 전혀 없는 1.0 사이의 값을 가집니다. 

p {background-color: gba(255, 165, 0, 0);}
p {background-color: gba(255, 165, 0, 0.5);}
p {background-color: gba(255, 165, 0, 1);}

orange

#2. HSL 색상값으로 표현

HSL 색상값은 빛의 삼원색으로 색을 표현하는 RGB 색상값과는 달리 색상, 채도, 명도를 사용해서 색을 표현합니다.
즉 HSL은 각각 색상(Hue), 채도(Saturation), 명도(Lightness)를 의미합니다.
색상은 0부터 360 사이의 값에 따라 결정되며, 색상환(color wheel)의 각도를 나타냅니다.

! 예시 !
<p style="background-color: hsl(39, 100%, 50%);">orange</p>

orange

#3. HSLA값으로 표현

rgba와 마찬가지로 hsl에 불투명도를 추가한 색상 표현 방법입니다.

! 예시 !
<p style="background-color: hsl(39, 100%, 50%, 0);">orange</p>
<p style="background-color: hsl(39, 100%, 50%, 0.5);">orange</p>
<p style="background-color: hsl(39, 100%, 50%, 1);">orange</p>

#4. Opacity 속성

opacity 속성은 색상에 대한 투명도를 설정해 줍니다.
opacity 속성값은 완전한 투명 상태인 0.0부터 투명도가 전혀 없는 1.0 사이의 값을 가집니다.

! 예시 !
<p style="background-color: rgb(255, 165, 0); opacity: 0.5">orange</p>
