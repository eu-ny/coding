#1. SCSS

코딩이 복잡해지게 되면 CSS는 유지보수가 어려워지고 가독성 또한 떨어지게 됩니다.
이러한 단점을 보완하고자 등장한 개념이 SCSS라고 합니다. 때문에 SCSS는 코드의 재활용성을 높이고, 가독성 증가시켜 개발의 효율성을 높여주는 장점을 가지고 있습니다!

✏️ SCSS

$font-stack: Helvetica, sans-serif;
$primary-color: #333;

body {
  font: 100% $font-stack;
  color: $primary-color;
}

SCSS에서 $는 변수를 뜻하는 기호 입니다. 때문에 반복적으로 사용할 속성은 변수($)로 속성 값을 지정해주어 재활용을 쉽게 할 수 있습니다.

✏️ CSS

body {
font: 100% Helvetica, sans-serif;
color: #333;
}

! $ 사용법 !
$변수명: 속성값;
